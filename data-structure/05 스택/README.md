- **용도**

  - 함수 호출 시 활성 레코드(지역변수, 파라미터, return address) 저장

    <br>

  - 괄호검사

    ```python
    # 괄호의 종류: [대괄호], {중괄호}, (소괄호)
    # 수도코드
    check_matching(expr) {
    while (입력 expr의 끝이 아니면) {
    	ch ← expr의 다음 글자
    	switch (ch)
    		case '(': case '[': case '{':
    		ch를 스택에 삽입
    		break
    		case ')': case ']': case ']':
    		if (스택이 비어 있으면)
    			then 오류
    		else 스택에서 open_ch를 꺼낸다
    			if (ch와 open_ch가 같은 짝이 아니면)
    				then 오류 보고
    		break
    }
    if (스택이 비어 있지 않으면)
    	then 오류
    }
    ```
